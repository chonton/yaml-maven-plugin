yaml:write

Full name:

org.honton.chas:yaml-maven-plugin:0.0.1:write

Description:

Write Properties

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: generate-resources.

Required Parameters

<filename> String - Yaml filename
User property is: yaml.filename.
<path> String - Path prefix
User property is: yaml.path.

Optional Parameters

<skip> boolean - Skip writing properties
User property is: yaml.skip.
